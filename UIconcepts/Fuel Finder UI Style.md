## Fuel Finder UI Style Guide

#ui #design #style #palette #fuel-finder

## Общая стилистика

Интерфейс выполнен в **тёплом кофейно-бежевом стиле** с мягким, аккуратным, почти "premium dashboard" настроением.  
Визуально стиль сочетает:

- тёмную кофейную рамку
    
- светлый тёплый внутренний фон
    
- скруглённые блоки
    
- мягкий контраст
    
- минималистичный clean UI
    
- тёплые акцентные кнопки
    
- зелёный статусный акцент
    

Стиль должен выглядеть:

- **чисто**
    
- **спокойно**
    
- **гармонично**
    
- **не слишком ярко**
    
- **не слишком плоско**
    
- **без лишней пестроты**
    

---

## Основная концепция

### Внешняя рамка

- Тёмно-коричневая
    
- Почти однотонная
    
- Без градиента
    
- Выглядит как кофейная / шоколадная рамка
    

### Внутренний фон

- Светлый
    
- Тёплый
    
- Бежево-кремовый
    
- Не белый и не серый
    

### Общий контраст

- Тёмная рамка снаружи
    
- Светлая рабочая зона внутри
    
- Вкладки и карточки на светлом фоне
    
- Активные элементы выделяются тёплым карамельно-коричневым цветом
    

---

## Палитра цветов

### 1. Active highlight / активное выделение

Основной акцентный цвет для активной вкладки, активной кнопки, selected state:

```txt
#AB7743
```

Описание:

- тёплый карамельно-коричневый
    
- основной цвет выделения
    
- именно его использовать для active tab / selected item
    

---

### 2. Outer dark border / внешняя тёмная рамка

Роль:

- основная толстая внешняя рамка макета
    
- контур больших контейнеров при необходимости
    

Рекомендуемый тон:

```txt
#3A2416
```

Можно использовать близкие оттенки:

```txt
#2F1D12
#402718
#4A2D1A
```

Важно:

- цвет должен быть **тёмно-коричневым**
    
- не уходить в чёрный
    
- не использовать градиент по рамке
    

---

### 3. Main inner background / внутренний основной фон

Роль:

- фон всей основной рабочей области
    
- фон карточек / панелей / больших контейнеров
    

Рекомендуемый тон:

```txt
#F4E8D8
```

Близкие допустимые варианты:

```txt
#F1E3D2
#EFE0CE
#EEDFCC
```

Описание:

- светлый тёплый беж
    
- мягкий кремовый оттенок
    
- не белоснежный
    

---

### 4. Secondary light surfaces / вторичный светлый фон

Роль:

- фон кнопок в неактивном состоянии
    
- фон вкладок
    
- фон полей
    
- фон внутренних карточек
    

Рекомендуемый тон:

```txt
#F7EBDD
```

Допустимые близкие:

```txt
#F5E9DC
#F6ECDD
#F3E6D8
```

---

### 5. Text color / основной текст

Роль:

- заголовки
    
- основной интерфейсный текст
    
- подписи
    

Рекомендуемый цвет:

```txt
#3A2416
```

Для вторичного текста:

```txt
#6A4A34
```

---

### 6. Green accent / зелёный акцент

Роль:

- зелёный статус
    
- зелёные индикаторы
    
- точка статуса / accent state
    

Оставить как в утверждённом концепте:

```txt
#57C84D
```

или близкий мягкий зелёный:

```txt
#5AC85A
#63C85D
```

Важно:

- зелёный используется **не как основной active color**
    
- зелёный — это **status accent**
    
- основной active highlight = **#AB7743**
    

---

### 7. Borders inside components / внутренние границы

Роль:

- контуры вкладок
    
- контуры карточек
    
- линии контейнеров
    

Рекомендуемый цвет:

```txt
#5B3923
```

Или мягче:

```txt
#6A4330
```

Важно:

- внутренние бордеры тёмные
    
- но не грубые
    
- не pure black, если это не нужно специально
    

---

## Формы и геометрия

### Скругления

Все border line должны быть округлённые.

Рекомендация:

- **border-radius: 25% +-**
    
- визуально это мягкие rounded corners
    
- ни острых углов, ни супер-круглых pill-only форм везде подряд
    

Для практики можно использовать:

- крупные блоки: `22px - 32px`
    
- кнопки / вкладки: `18px - 24px`
    
- внутренние карточки: `16px - 20px`

## Компоненты

## Шапка / Header

Содержит:

- логотип / иконку топлива
    
- название `Fuel Finder header #1`
    
- короткий subtitle
    
- блок `My credentials`
    
- блок `version / db indicator / updated time stamp`
    

Стиль:

- светлые панели на тёмной рамке
    
- крупные скругления
    
- аккуратные тёмные контуры
    

---

## Навигационные вкладки

Разделы:

- Search bar
    
- Results
    
- Map
    
- Analytics/Trends
    
- All stations
    

Состояния:

- **inactive**: светлая кнопка, тёмный текст, тёмная рамка
    
- **active**: фон `#AB7743`, светлый текст или контрастный тёмный по необходимости
    

---

## Контентная область

Большой внутренний контейнер:

- светлый фон
    
- тёмная рамка
    
- мягкие углы
    
- много воздуха
    
- аккуратное расположение карточек, таблиц, графиков или карты
    

---

## Кнопки

### Primary button

Использовать:

```txt
background: #AB7743
text: light / cream / white-ish
border: dark brown
```

### Secondary button

Использовать:

```txt
background: #F4E8D8
text: #3A2416
border: #5B3923
```

---

## Карточки и таблицы

Карточки станций / карточки результатов:

- светлый бежевый фон
    
- тёмно-коричневый текст
    
- мягкий контур
    
- лёгкая аккуратная иерархия по заголовкам и значениям
    

---

## Типографика

### Заголовки

Стиль:

- крупные
    
- уверенные
    
- тёмно-коричневые
    
- могут быть чуть более "serif-like" или semi-bold по настроению, если нужен более elegant вид
    

### Основной текст

Стиль:

- простой
    
- читаемый
    
- без лишнего декоративного шума
    
- цвет тёмно-коричневый
    

### Подписи / secondary text

Стиль:

- спокойный
    
- чуть светлее основного текста
    
- без сильного контраста
    

---

## Что важно не делать

- не использовать холодные серо-синие оттенки
    
- не делать чёрный как основной цвет фона
    
- не использовать яркий кислотный оранжевый вместо `#AB7743`
    
- не делать белоснежный фон
    
- не использовать градиент по внешней рамке
    
- не делать слишком резкие sharp corners
    
- не перегружать UI лишними цветами
